No.011 告白 - 200

全ての人に振られる確率なので、積の法則を使うことができます。

積の法則が分からない人へ 積の法則とは、
「2つ以上の事象が共に起こるとき、その確率は個々の事象が起こる確率の積になる」
ということです。
これだけでは分かりにくいと思うので具体例を挙げます。
例えば、6面のサイコロを振り、3回連続で1が出る確率を求めたいとき、
$\frac{1}{6} \times \frac{1}{6} \times \frac{1}{6}$
で求めることができます。
これはサイコロを1回振った時に1が出る確率が$\frac{1}{6}$であるため、それを3回掛けることによって求めることができるからです。

一回告白したときに振られる確率は問題文より$\frac{1}{2}$です。
これをperson回行うため、求める確率は$\frac{1}{2}^{person}$であると分かります。
最後に百分率に直すのを忘れないようにしましょう。

実装例

person = 2

print((0.5 ** person) * 100)

No.016 土地の値段 - 200

解法1

2020年に土地を買っているので、(year - 2020)年間土地を保有していたことになります。
for文で実際にシミュレートすることによって簡単にこの問題を解くことができます。

実装例1

year = 2022

ans = 1
n = year - 2020

for i in range(n):
    ans *= 2

print(ans)

解法2

for文を使わなくても解くことができます。
保有していた年数分2を掛ければよいので、以下のように実装することで解くことができます。
この問題では存在しませんが、yearの値が非常に大きかった場合、for文では処理に時間がかかりすぎて終わらないということも考えられるので、なるべく繰り返しを避けて問題を解くことができないか考えることも重要です。

実装例2

year = 2022

n = year - 2020

print(2 ** n)

No.023 チョコボール - 200

for文を使って実際に食べたチョコボールの数をシミュレートしてみましょう。
食べたチョコボールの数が持っているチョコボールの数を超える日がチョコボールがなくなる日になります。

実装例

choco_ball = 20

sum = 0 # 食べたチョコボールの数を表す変数
day = 1 # 日数を数える変数

while True:
    sum += day # 食べる個数は食べる日の数と同じなのでdayを足す
    if sum >= choco_ball: # 食べたチョコボールの数が持っているチョコボールの数を超えたらループを抜ける
        break
    day += 1 # 日数を1増やす

print(day)

No.184 何通り？ - 500

No.220 Coprime - 500

誤答例

単純な解法としては実際に$A^B$、$B^A$を計算し、
その最大公約数が1かどうか判定する方法が考えられます。
この場合、次のようなコードになります。

import math

A = 2
B = 5

if math.gcd(pow(A, B), pow(B, A)) == 1:
    print("Yes")
else:
    print("No")

「なんだ、これだけで解けるんじゃないか」と思うかもしれませんが、これで答え合わせをしてみると… 長い時間待っても答え合わせが終わりません。 何が起こっているのでしょうか。

出てくる数字について

この問題では制約が$1 \leqq A, B \leqq 10^9$となっています。
この場合、$A^B$を計算する時に最大で$10^{9 \times 10^9}$という途方もない数値になってしまいます。
トイプロでは処理に5秒以上かかる場合、答え合わせが中断されるようになっているため、この解法では問題を解けないということになります。

想定解

「$A^B$と$B^A$が互いに素であるか」という問題を言い換えることはできないでしょうか？ 実は、互いに素である整数の性質として、以下のようなものがあります。

$A, Bが整数であり、n, mが自然数のとき、$ $AとBが互いに素であるならばA^nとB^mも互いに素である$ $また、逆にA^nとB^mが互いに素であるならば、AとBも互いに素である$

これはAとBが互いに素である場合、共通の素因数を持っていないということであり、
ある数字をべき乗してもその素因数の種類は変わらないという性質から導くことができます。

さて、以上のことを踏まえると、$A^B$と$B^A$が互いに素か判定するには$A$と$B$が互いに素か判定すれば良いということになります。

実装例

import math

A = 2
B = 5

if math.gcd(A, B) == 1:
    print("Yes")
else:
    print("No")
